🤖 AI Builder

AI Builder cho phép thêm AI capabilities vào Power Apps và Power Automate mà không cần kiến thức machine learning:

📄

Document Processing

Trích xuất data từ invoices, receipts, business cards tự động.

📝

Text Analysis

Sentiment analysis, key phrase extraction, language detection.

📊

Prediction

Dự đoán outcomes dựa trên historical data (churn, default...).

👁️

Object Detection

Nhận diện objects trong hình ảnh. Sử dụng cho QC, inventory.

📋

Category Classification

Tự động phân loại text vào categories (support tickets, emails).

💬

GPT Models

Tích hợp Azure OpenAI / GPT cho text generation, summarization.

Ví dụ: Invoice Processing Flow
TRIGGER: When email arrives with attachment
  ↓ 
ACTION: AI Builder - Extract info from invoices
  → Input: Email attachment
  ↓
ACTION: Create item in SharePoint
  → Vendor: AI output → Vendor name
  → Amount: AI output → Total amount
  → Date: AI output → Invoice date
  → Invoice #: AI output → Invoice number

📊 Power BI Integration

Power Platform tích hợp chặt chẽ với Power BI cho báo cáo và phân tích:

  • Embed Power BI in Canvas Apps: Nhúng reports/tiles trực tiếp vào app
  • Embed in Model-driven: Power BI dashboards trong navigation
  • Dataverse as data source: Power BI connect trực tiếp tới Dataverse
  • Power Automate triggers: Trigger flows khi Power BI alerts fire
  • Row-level security: Data security từ Dataverse respects trong BI

🔄 Application Lifecycle Management (ALM)

Quản lý vòng đời ứng dụng Power Platform một cách chuyên nghiệp:

Development Environment

Develop & test trong Dev environment. Sử dụng unmanaged solutions.

Source Control

Export solution → Unpack với Power Platform CLI → Commit to Git (Azure DevOps/GitHub).

Build & Test

CI pipeline: build managed solution, run automated tests, validate.

Deploy

CD pipeline: import managed solution vào Test → UAT → Production environments.

Environment Strategy

Environment Architecture

┌───────────────┐       ┌───────────────┐       ┌───────────────┐

⚡ Performance Optimization

Canvas Apps

  • Delegation: Sử dụng delegable functions (Filter, LookUp) thay vì non-delegable (Search, CountRows)
  • Concurrent(): Load nhiều data sources song song khi app start
  • Collections: Cache data thường dùng bằng ClearCollect() thay vì query trực tiếp
  • Minimize controls: Giảm số controls trên mỗi screen (< 300)
  • Delay loading: Dùng Timer hoặc variable để delay load heavy content

Power Automate Flows

  • Parallel branches: Chạy actions song song khi không phụ thuộc nhau
  • Concurrency control: Bật cho Apply to each (max 50 parallel)
  • Select & Filter: Sử dụng Select action thay vì Apply to each cho data transformation
  • Minimize actions: Gộp logic vào expressions thay vì nhiều actions
  • Child flows: Chia flow lớn thành child flows để đọc và maintain dễ hơn

🤖 Copilot Studio (Power Virtual Agents)

Tạo AI chatbots không cần code, tích hợp với Power Platform:

  • Topic-based design: Thiết kế conversation flows theo topics
  • Generative AI answers: Bot trả lời dựa trên knowledge sources (websites, docs)
  • Power Automate integration: Gọi flows từ bot để thực hiện actions
  • Multi-channel: Deploy trên Teams, websites, Facebook, custom channels
  • Analytics: Dashboard chi tiết về user interactions, satisfaction, escalations
