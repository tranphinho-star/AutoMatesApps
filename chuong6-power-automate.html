⚙️ Power Automate là gì?

Power Automate (trước đây là Microsoft Flow) là dịch vụ tự động hóa quy trình kinh doanh — cho phép tạo automated workflows giữa các ứng dụng và dịch vụ để đồng bộ files, nhận notifications, thu thập dữ liệu, v.v.

🔄

Tự động hóa

Giảm công việc thủ công, lặp đi lặp lại. Tập trung vào việc quan trọng hơn.

🔌

400+ Connectors

Kết nối Microsoft services, third-party apps, on-premises systems.

🖥️

RPA (Desktop)

Tự động hóa desktop applications — web scraping, data entry, legacy systems.

🤖

AI Builder

Tích hợp AI: form processing, object detection, text classification, prediction.

💡 Power Automate vs Traditional Coding
Power Automate cho phép business users tạo automations mà không cần viết code. Tuy nhiên, IT professionals cũng có thể sử dụng để xây dựng enterprise-grade workflows phức tạp.

📊 Các loại Flows

Power Automate cung cấp 3 loại flows chính:

☁️

Cloud Flows

Chạy trên cloud. 3 sub-types:
• Automated: trigger bởi event
• Scheduled: chạy theo lịch
• Instant: kích hoạt thủ công

🖥️

Desktop Flows

RPA — tự động hóa ứng dụng desktop. Record & playback UI actions. Chạy trên máy local hoặc VM.

📋

Business Process Flows

Hướng dẫn users qua quy trình nghiệp vụ từng bước. Đảm bảo consistency & compliance.

🎯 Triggers & Actions

Mỗi flow gồm 2 thành phần chính:

🎯 Triggers

  • Sự kiện khởi chạy flow
  • Mỗi flow có đúng 1 trigger
  • Ví dụ: "Khi item mới được tạo"
  • Tần suất polling: ~1-5 phút

⚡ Actions

  • Hành động thực hiện sau trigger
  • Có thể có nhiều actions
  • Ví dụ: "Gửi email", "Tạo record"
  • Chạy tuần tự hoặc song song

Ví dụ flow đơn giản

Flow: Thông báo khi có item SharePoint mới
TRIGGER: When an item is created (SharePoint)
  ↓
ACTION 1: Get item details (SharePoint)
  ↓
ACTION 2: Send an email (Outlook)
  - To: manager@company.com
  - Subject: "[New] " + Title
  - Body: "Item mới: " + Description
  ↓
ACTION 3: Post message (Teams)
  - Channel: #notifications
  - Message: "📢 New item: " + Title

🔍 Process Advisor

Process Advisor giúp bạn phân tích và tối ưu hóa quy trình kinh doanh trước khi tự động hóa:

🖱️

Task Mining

Record hoạt động của users trên desktop. Phân tích patterns, bottlenecks, và cơ hội automation.

📊

Process Mining

Import event logs từ hệ thống. Tạo process maps, phát hiện deviations, và tối ưu hóa workflows.

Record Processes

Users record các tác vụ hàng ngày sử dụng recorder tool.

Analyze

Process Advisor phân tích recordings, tìm patterns và bottlenecks.

Recommend

Đề xuất automations phù hợp: cloud flows, desktop flows, hoặc kết hợp.

Automate

Tạo flows trực tiếp từ recommendations.

🚀 Getting Started

Truy cập Power Automate

Mở flow.microsoft.com hoặc make.powerautomate.com.

Đăng nhập

Sử dụng tài khoản Work hoặc School. Power Automate đi kèm với nhiều Microsoft 365 plans.

Chọn Template hoặc Blank

Bắt đầu từ template phổ biến hoặc tạo flow mới từ blank.

Cấu hình & Test

Thêm trigger, actions, test flow, và bật "Turn on" để activate.
