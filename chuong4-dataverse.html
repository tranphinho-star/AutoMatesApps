🗄️ Dataverse là gì?

Microsoft Dataverse (trước đây: Common Data Service - CDS) là nền tảng dữ liệu được thiết kế riêng cho Power Platform. Nó cung cấp khả năng lưu trữ, quản lý, và bảo mật dữ liệu kinh doanh một cách chuẩn hóa.

☁️

Cloud-Native

Lưu trữ trên Azure cloud. Tự động scaling, backup, and disaster recovery.

📊

Standard Tables

Sẵn có 400+ standard tables: Account, Contact, Activity, Case, etc.

🔒

Enterprise Security

Role-based access control, field-level security, audit trails.

🔗

Integration

Tích hợp Power BI, Power Automate, Dynamics 365, Azure services.

💡 Tại sao chọn Dataverse?

✅ Ưu điểm Dataverse

  • Bảo mật enterprise-grade
  • Business logic tích hợp
  • Standard data model
  • API sẵn có (Web API, OData)
  • Relationships & validations
  • Audit & change tracking
  • Offline support

⚡ So với SharePoint/Excel

  • SharePoint: tốt cho docs, hạn chế cho data
  • Excel: giới hạn records, no security
  • Dataverse: designed for data management
  • Complex data types support
  • Better performance at scale
  • Server-side business logic
  • Yêu cầu Premium license

📋 Tables (Bảng dữ liệu)

Tables là đơn vị lưu trữ cơ bản trong Dataverse. Có 3 loại tables:

Loại TableMô tảVí dụ
StandardCó sẵn trong Dataverse, cấu trúc chuẩnAccount, Contact, Activity, Task
CustomTạo mới theo nhu cầu riêngProducts, Projects, Invoices
VirtualMaps to external data (không lưu trong Dataverse)SQL tables, SharePoint lists

Tạo Custom Table

Mở Tables

make.powerapps.com → Tables → New table

Đặt tên & cấu hình

Display name, plural name, primary column name. Chọn table type & ownership.

Thêm Columns

Thêm các cột dữ liệu: text, number, date, lookup, choice, etc.

Lưu & Sử dụng

Save table. Có thể sử dụng ngay trong Power Apps, Power Automate.

📝 Columns (Cột dữ liệu)

Mỗi table chứa nhiều columns. Các loại column phổ biến:

Loại ColumnMô tảVí dụ
TextChuỗi ký tự (max 4000 chars)Name, Email, Address
Whole NumberSố nguyênQuantity, Age
DecimalSố thập phânPrice, Weight
CurrencyGiá trị tiền tệ (auto format)Total, Salary
Date/TimeNgày giờCreated On, Due Date
ChoiceDanh sách lựa chọn (enum)Status, Priority, Category
Yes/NoBooleanIsActive, IsCompleted
LookupTham chiếu đến record khácCustomer → Account
File/ImageFile đính kèm hoặc hình ảnhPhoto, Document
FormulaTính toán từ columns khácFullName = First + " " + Last

🔗 Relationships (Quan hệ)

Dataverse hỗ trợ 3 loại relationships giữa tables:

1️⃣

One-to-Many (1:N)

Một Account có nhiều Contacts. Phổ biến nhất. Tạo bằng Lookup column.

🔁

Many-to-One (N:1)

Ngược lại của 1:N. Nhiều Orders thuộc về 1 Customer.

🔀

Many-to-Many (N:N)

Nhiều Students enroll nhiều Courses. Tạo intersect table tự động.

📝 Cascade behaviors
Khi thiết lập relationships, cần chỉ định cascade behavior: khi xóa parent record thì child records sẽ bị xóa (Cascade), giữ nguyên (Remove Link), hoặc chặn xóa (Restrict).

⚙️ Business Logic

Dataverse hỗ trợ nhiều cách để thêm logic nghiệp vụ:

  • Business Rules: Validation, set values, show/hide fields — no code
  • Calculated Columns: Tính toán giá trị từ columns khác
  • Rollup Columns: Aggregate dữ liệu từ related records (SUM, COUNT, AVG)
  • Power Automate Flows: Triggers khi record created/updated/deleted
  • Plugins: Custom code (C#) cho logic phức tạp — dành cho developers
  • Classic Workflows: Background workflows kế thừa từ Dynamics

📦 Solutions

Solutions là cơ chế đóng gói tất cả components của ứng dụng (tables, apps, flows, security roles) để quản lý, export và deploy sang environments khác.

📦 Managed Solutions

  • Không thể chỉnh sửa trực tiếp
  • Dùng cho production deployment
  • Có thể uninstall hoàn toàn
  • Versioned

🔓 Unmanaged Solutions

  • Có thể chỉnh sửa tự do
  • Dùng cho development
  • Components merge vào default solution
  • Export thành managed để deploy

🔒 Security & Roles

Dataverse cung cấp bảo mật nhiều tầng:

  • Security Roles: Định nghĩa quyền CRUD trên từng table (Organization, Business Unit, User level)
  • Field-Level Security: Kiểm soát quyền truy cập từng column
  • Record-Level Security: Sharing, teams, access teams
  • Hierarchy Security: Manager hierarchy tự động
  • Audit: Ghi lại tất cả thay đổi dữ liệu cho compliance
💡 License
Dataverse yêu cầu Premium license (Power Apps per user hoặc per app plan). Tuy nhiên, Dataverse for Teams (giới hạn) đi kèm với license Microsoft 365.
