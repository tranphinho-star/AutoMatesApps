🎨 Canvas Apps là gì?

Canvas Apps cho phép bạn tạo ứng dụng bằng cách kéo thả elements trên một "canvas" trống — giống như thiết kế slide trong PowerPoint. Bạn có toàn quyền kiểm soát giao diện người dùng.

✏️

Pixel-Perfect Design

Tự do sắp xếp mọi element trên canvas. Kiểm soát kích thước, vị trí, màu sắc của từng control.

📱

Multi-Platform

Ứng dụng chạy trên web browser, iOS, Android và Windows. Một lần build, chạy mọi nơi.

🔌

200+ Connectors

Kết nối với SharePoint, Excel, SQL, Salesforce, và hàng trăm service khác.

⚡

Rapid Development

Tạo ứng dụng CRUD hoàn chỉnh trong vài phút từ data source có sẵn.

🚀 Tạo app đầu tiên

Có nhiều cách để bắt đầu tạo Canvas App:

Cách 1: Tạo từ Data Source (Nhanh nhất)

Chuẩn bị Data Source

Tạo SharePoint list hoặc Excel table với dữ liệu mẫu. Ví dụ: danh sách "Consulting Interventions" trên SharePoint.

Mở Power Apps → Create an App

Truy cập make.powerapps.com, chọn "Create", sau đó chọn "Start from data".

Chọn Data Source & Connect

Chọn connector (SharePoint, Excel, SQL...), sau đó chọn table/list cụ thể.

App được tạo tự động!

Power Apps Studio mở ra với ứng dụng 3 screens hoàn chỉnh: Browse, Detail, Edit. Bạn có thể tùy chỉnh từ đây.

Cách 2: Tạo từ Template

Power Apps cung cấp nhiều template sẵn có cho các kịch bản phổ biến:

  • Budget Tracker — Theo dõi ngân sách
  • Leave Request — Đơn xin nghỉ phép
  • Help Desk — Hệ thống hỗ trợ IT
  • Site Inspection — Kiểm tra hiện trường
  • Inventory Management — Quản lý kho

Cách 3: Tạo từ Blank Canvas

Bắt đầu với canvas trống hoàn toàn. Phù hợp khi bạn muốn kiểm soát tối đa thiết kế.

💡 Khuyến nghị cho người mới
Hãy bắt đầu với Cách 1 (từ Data Source) để hiểu cấu trúc cơ bản. Sau đó thử tùy chỉnh app được tạo tự động để học về controls và formulas.

🖥️ PowerApps Studio & Designer

PowerApps Studio là môi trường thiết kế chính, gồm 3 panels và ribbon:

📐

Left Panel - Tree View

Hiển thị cấu trúc cây của tất cả screens và controls. Click để chọn, kéo thả để sắp xếp.

🎨

Center - Canvas

Vùng thiết kế chính. Kéo thả controls, sắp xếp layout, preview giao diện trực tiếp.

⚙️

Right Panel - Properties

Cấu hình properties của control đang chọn: size, color, data bindings, formulas.

📝

Formula Bar

Thanh công thức ở trên cùng — viết Power Fx formulas cho mỗi property của control.

📝 Ribbon tabs quan trọng
  • Home: Insert controls, undo/redo, clipboard
  • Insert: Thêm controls (buttons, labels, galleries, forms...)
  • View: Data sources, collections, variables
  • Action: Flows, navigation, notifications
  • File: Save, publish, app settings

📱 Screens & Controls

Screens — Màn hình ứng dụng

Mỗi Canvas App gồm một hoặc nhiều screens. Mỗi screen là một trang riêng biệt chứa các controls.

  • BrowseScreen: Hiển thị danh sách records dạng gallery
  • DetailScreen: Hiển thị chi tiết một record
  • EditScreen: Biểu mẫu tạo mới / chỉnh sửa record
  • StartScreen: Trang welcome / menu chính

Controls — Thành phần giao diện

Controls là các building blocks cơ bản của Canvas App. Các nhóm controls quan trọng:

Nhóm Control Ví dụ Mô tả
Input Text Input, Dropdown, Date Picker, Toggle, Slider Nhận dữ liệu từ người dùng
Display Label, HTML Text, Image, Video, PDF Viewer Hiển thị thông tin
Gallery Vertical, Horizontal, Blank Gallery Hiển thị danh sách records
Form Edit Form, Display Form Tạo/sửa/xem records từ data source
Layout Container, Horizontal Container, Scrollable Screen Sắp xếp layout responsive
Action Button, Timer, Power Automate Thực hiện hành động khi click/trigger
Charts Column, Line, Pie Chart Biểu đồ trực quan
Media Camera, Barcode Scanner, Audio, Microphone Tương tác phần cứng thiết bị
Icons Add, Edit, Delete, Search, Check icons Biểu tượng cho actions
💡 Gallery là control quan trọng nhất
Gallery kết nối trực tiếp với data source và tự động hiển thị danh sách records. Mỗi item trong gallery là một template có thể chứa nhiều controls con (label, image, icon...).

📐 Power Fx - Ngôn ngữ công thức

Power Fx là ngôn ngữ công thức của Power Apps, lấy cảm hứng từ Excel formulas. Nếu bạn biết dùng Excel, bạn sẽ học Power Fx rất nhanh!

Formulas cơ bản

Power Fx — Ví dụ cơ bản
// Điều hướng tới screen khác
Navigate(DetailScreen, ScreenTransition.Fade)

// Lọc dữ liệu
Filter(Employees, Department = "IT")

// Sắp xếp
SortByColumns(Employees, "Name", SortOrder.Ascending)

// Tìm kiếm
Search(Products, SearchBox.Text, "Name", "Category")

// Đếm records
CountRows(Filter(Tasks, Status = "Completed"))

Formulas thao tác dữ liệu

Power Fx — CRUD Operations
// Tạo record mới (Create)
Patch(Employees, Defaults(Employees), {
    Name: txtName.Text,
    Email: txtEmail.Text,
    Department: ddDepartment.Selected.Value
})

// Cập nhật record (Update)
Patch(Employees, Gallery1.Selected, {
    Status: "Active"
})

// Xóa record (Delete)
Remove(Employees, Gallery1.Selected)

// Submit form
SubmitForm(EditForm1)

// Reset form
ResetForm(EditForm1)

Functions phổ biến

Function Mô tả Ví dụ
Navigate()Chuyển screenNavigate(Screen2)
Filter()Lọc dữ liệuFilter(Items, Price > 100)
Search()Tìm kiếm textSearch(Items, "laptop", "Name")
Patch()Tạo/sửa recordPatch(Table, record, changes)

🔌 Kết nối Data Sources

Trong Canvas Apps, bạn kết nối với data sources qua connectors:

Mở Data Panel

Trong Studio, click View → Data sources hoặc icon Data bên trái.

Add Data Source

Click "Add data", tìm connector mong muốn (SharePoint, Excel, SQL...).

Authenticate & Select

Đăng nhập (nếu cần), chọn site → list/table cụ thể để kết nối.

Sử dụng trong Formulas

Data source xuất hiện trong formulas: Filter(MyList, Status="Active")

⚠️ Delegation
Power Apps có giới hạn delegation — chỉ tải tối đa 500-2000 records mặc định. Với data sources lớn, sử dụng Filter() thay vì Search() và chọn operations được delegate (=, <>, StartsWith...).

▶️ Preview, Save & Publish

👁️

Preview (F5)

Nhấn F5 hoặc icon ▶ để test app trực tiếp trong Studio. Nhấn Esc để quay lại edit.

💾

Save (Ctrl+S)

Lưu version mới. Power Apps tự động lưu version history, bạn có thể rollback bất kỳ lúc nào.

📤

Publish

Xuất bản app cho users. Chỉ published version mới visible cho người được chia sẻ. File → Publish.

👥 Share & Manage

Sau khi publish, bạn cần share app cho người dùng:

  • Share with specific users/groups: Nhập email hoặc security group
  • Co-owner: Người được share có thể edit app (nếu bạn cho quyền)
  • User: Chỉ có quyền sử dụng app, không chỉnh sửa
  • Data permissions: Đảm bảo users có quyền truy cập data source
💡 Quan trọng
Khi share app, bạn cần đảm bảo người dùng cũng có quyền truy cập data sources mà app sử dụng. Ví dụ: nếu app dùng SharePoint list, user cần có permission trên list đó.
