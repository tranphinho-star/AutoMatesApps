🔗 Tổng quan Connectors

Connectors là cầu nối giữa Power Apps/Power Automate và các data sources bên ngoài. Mỗi connector đóng gói một REST API, cho phép bạn truy cập dữ liệu và thực hiện actions mà không cần viết code API.

Connectors được chia sẻ giữa Power Apps, Power Automate, và Logic Apps — tạo connector một lần, dùng mọi nơi.

📋

Tables

Truy xuất dữ liệu dạng bảng — CRUD operations trên records.

⚡

Actions

Thực hiện hành động cụ thể — gửi email, approve, trigger flow.

🔔

Triggers

Khởi chạy flow khi có sự kiện — new item, modified record.

🛠️ Custom Connectors

Tạo connector riêng cho API của bạn hoặc API bên thứ 3 chưa có connector sẵn:

Chuẩn bị API Definition

Cần OpenAPI (Swagger) definition, Postman collection, hoặc tạo thủ công.

Tạo Custom Connector

make.powerapps.com → Data → Custom Connectors → New. Import definition hoặc tạo từ blank.

Cấu hình Authentication

Chọn phương thức: API Key, OAuth 2.0, Basic Auth, hoặc No Authentication.

Define Actions & Triggers

Thêm các actions (GET, POST, PUT, DELETE) và cấu hình request/response schema.

Test & Publish

Test connector, sau đó sử dụng trong Power Apps và Power Automate như connector thường.

💡 Certify & Share
Bạn có thể certify custom connector với Microsoft để share công khai cho cộng đồng. Connector certified sẽ xuất hiện trong danh sách connectors chính thức.

🏢 On-premises Data Gateway

Gateway cho phép kết nối an toàn từ cloud tới data sources on-premises:

Cách hoạt động

  • Cài đặt Gateway application trên server on-premises
  • Gateway tạo kênh encrypted qua Azure Service Bus
  • Power Apps/Automate gửi request → Gateway → On-premises data → Trả kết quả
  • Gateway chia sẻ giữa: Power BI, Power Apps, Power Automate, Azure Logic Apps

🔧 Quản lý Connections

Connections là instances cụ thể của connectors với credentials đã lưu:

  • Tạo connection: Khi thêm data source vào app lần đầu
  • Quản lý: make.powerapps.com → Data → Connections
  • Shared connections: Chia sẻ connection với users khác (họ dùng credentials riêng)
  • Connection references: Sử dụng trong solutions để portable giữa environments

📐 Data Integration Patterns

📊

Direct Connection

App kết nối trực tiếp tới data source. Đơn giản nhất — data luôn real-time.

💾

Collections (Cache)

Load data vào collection cục bộ bằng ClearCollect(). Giảm API calls, tăng performance.

🔄

Dataflows

ETL process: Extract data từ sources → Transform → Load vào Dataverse. Scheduled refresh.

🔀

Dual-Write

Sync real-time giữa Dataverse và Finance/Operations. Cho enterprise integration.
